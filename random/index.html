Let's get a random number between 0 and 10

Math.floor(Math.random() * 10)

(Math.random()) => finds a random number between 0 and .999999


(Math.random() * 10) => takes a random number and multiplies it by 10 in this example


Math.floor(Math.random() * 10) => returns a random integer between 0 and 9 (both included):
After multiplying rounds down to the nearest whole number, cut off the decimal places


Let's try this again with a number between 100 and 200

Math.floor(Math.random() * 100) + 100

(Math.random()) => finds a random number between 0 and .999999


(Math.random() * 100); => takes a random number and multiplies it by 100 in this example

100 was used because you take (max - min), in this case (200 -100)


Math.floor(Math.random() * 100) + 100; => returns a random number between 100 and 200 then adds 100 for the starting number

After multiplying rounds down to the nearest whole number, cut off the decimal places
